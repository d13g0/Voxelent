Rendering a Brain Mesh
Your browser does not support the HTML5 canvas element.

Mouse Interaction

Rotate the brain by dragging the left mouse button over the canvas. You can zoom in and out by holding the Ctrl key pressed while you drag your mouse

Multiple representations

  • Open the JavaScript console. In Firefox press Ctrl+Shift+K, in Chrome Ctrl+Shift+J
  • Try vxl.api.wireframeON()
  • Enter vxl.api.surfaceON() to go back to surface mode
  • Now try vxl.api.pointsON(). What do you see?

Changing the Lookup Table

  • Open the JavaScript console. In Firefox press Ctrl+Shift+K, in Chrome Ctrl+Shift+J
  • Enter vxl.api.getLookupTables(). Voxelent will return the list of lookup tables that are loaded
  • Enter vxl.api.setLookupTable('cortex')
