Class Detail

vxlCamera(vw, t)
A vxlCamera object simplifies WebGL programming by providing a simple object interface to the lower level matrix manipulations that are required to view a 3D scene. When moving and rotating the camera, such matrices are updated and Voxelent's Nucleo will use this information in order to draw the scene accordingly to the camera position an orientation. A vxlCamera object requires a vxlView to be created. Having said that, a vxlView can be associated with multiple cameras.
Author: Diego Cantor.

Parameters:

{vxlView} vw

{Object} t

the type of camera

void dolly(value)
Performs the dollying operation in the direction indicated by the camera normal axis. The dollying mechanism offered by a camera makes sure that the camera moves fast towards the object when the distance is large and slow when it is very close to the object. For that effect, every time that the new position (after dollying) is calculated, the field dstep is computed.

Parameters:

{Number} value

the dollying value
